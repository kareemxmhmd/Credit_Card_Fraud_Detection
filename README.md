# Credit Card Fraud Detection

## Overview

Main project files:

- `EDA.ipynb`: exploratory data analysis.
- `ML_FD.py`: full machine learning training and evaluation pipeline.
- `Data/creditcard.csv`: transaction dataset.
- `Models/Random Forest_model.pkl`: saved best model.
- `Models/scaler.pkl`: saved scaler for `Amount` and `Time`.
- `Visualizations/`: confusion matrices, ROC/PR curves, threshold tuning plot, and SHAP plots.
